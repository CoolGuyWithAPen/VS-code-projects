# Set Status
This cog for Red Discord Bot has one task, and that is to give you an easy, but simple slash command to use to set a custom status for your Discord bot!

## Installation

```
[p]repo add CoolGuyWithAPen https://github.com/CoolGuyWithAPen/VS-code-projects
[p]cog install CoolGuyWithAPen setstatus
[p]load setstatus
[p]slash enable setstatus
```

## Usage

```/setstatus text:Howdy!```
Now your discord bot has a custom status "Howdy!"
